Programming w/ Ladder Logic
Languages: Ladder Logic

Programs/Tech: RSLogix500, RSLogix 5000, Studio5000, PanelView

Description: I worked for 3.5 years writing programmable logic controller (PLC) code. PLC programming involves a langauge called, "ladder logic". It's a way to visualize the electrical contacts, relays, coils, etc. and write software to manipulate said components. Ladder logic is used in factory automation and is similar to any other programming language from an algorithm standpoint.

SSH Keys & Git to Deploy Code
Programs/Tech: Git, PuTTY

Description: The purpose of this project was to make my workflow more efficient and streamlined. In short, the process is as follows:
  1. Create public/private SSH keys.
  2. Link SSH keys between local development machine and web server.
  3. Link SSH key to Github.
  4. Create local and remote repositories.
  5. Create post-recieve hook to checkout a branch.
  6. Use Git remotes to push code to both Github and web server simultaneously by using: git add, git commit and git push.
Lessons Learned:
  1. How to create SSH keys.
  2. How to link SSH keys between machines.
  3. How to use Git with a command line interface (CLI).
  4. How to deploy code with Gi.
  5. How to improve and streamline my workflow.
